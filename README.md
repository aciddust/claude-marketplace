# Claude Marketplace PoC

마켓플레이스가 무엇인지 알아보려고 만들어본 레파지토리

## 설치

```bash
# 마켓플레이스 등록 (초기 1회 진행)
claude plugin marketplace add aciddust/claude-marketplace

# 플러그인 설치 예시
claude plugin install helper@d3fau1t
```

## 플러그인 목록

### helper

- `/setup` 커맨드를 오버라이드합니다.
- 프로젝트 셋업, 분석시 초기 설정을 여러단계로 나누어 사용자에게 물어보고 조율합니다.

## 업데이트

```bash
## 마켓플레이스 목록 갱신
claude plugin marketplace update aciddust
```
